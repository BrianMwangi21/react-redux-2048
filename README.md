## Setup

```
git clone git@github.com:CeRRuTiTo/react-redux-starter.git
cd react-redux-starter
npm install
```


## Usage

### `start`
Start the server:
```
make start
```

### `test`
Run the tests:
```
make test
```

### `watch`
Run and watch the tests:
```
make watch
```


## Warnings

- Soft warning when using LogMonitor
